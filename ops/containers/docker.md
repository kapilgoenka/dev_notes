# Docker

## Dev Containers

### Concept

* Allow you to set up a development environment and tooling within a docker container and interact with that container via your IDE, just as you would a workspace hosted on your own laptop.

* The main benefit is there’s no need to install tooling on your own laptop, and this makes it easy for new starters to start working on a project.

* A prerequisite is to have docker (CLI) installed.

### VSCode / Cursor Extension

* Provided by Microsoft

* Lets you use a Docker container as a full-featured development environment.
